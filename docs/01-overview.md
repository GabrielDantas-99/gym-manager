# Sistema Gerenciador de Academias

## Objetivo

Sistema SaaS para gestão completa de academias, permitindo:

- Gestão de usuários (ADMIN, PERSONAL, ALUNO)
- Gestão de academias
- Vínculo entre usuários
- Gestão de treinos
- Controle financeiro
- Dashboard administrativo

## Stack Tecnológica

### Backend

- Java 17
- Spring Boot 3.2.1
- Spring Security + JWT
- Spring Data JPA
- H2 (dev)
- PostgreSQL (prod)
- Docker

### Frontend

- Vite.js
- React (definição recomendada)
- TypeScript
- Axios
- React Router
- Context API ou Zustand

## Status Atual

- Autenticação implementada
- Controle de roles implementado
- Gestão de usuários e vínculos implementada
- Módulos de Treino, Financeiro e Dashboard em desenvolvimento
